# umi实践问题集锦

## 1.webpackChain自定义loader报错：this.getOptions is not a function

**场景：**
    添加loader解析makdown文件转换为html展示，使用webpackChain增加remark-loader,html-loader报错。
```
export default {
    ...
    chainWebpack(config){
        config.module
        .rule('compile-md')
        .test(/\.md$/)
            .use("html-loader")
            .loader("html-loader")
            .end()
            .use("markdown-loader")
    }
    ...
}
  
```
**解决：**
    关闭webpack5编辑即可，修改umi配置config.js 或 .umirc.js：
```
export default {
    ...
    webpack5:false,
    ...
}
```
